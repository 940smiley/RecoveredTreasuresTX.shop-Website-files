Highlighted Capabilities

AI-Ready Pipelines

Slots reserved for OpenAI, Google Vision, and Cloudinary to power description generation, quality scoring, and enhancement.

Gallery & Uploads

Showcase browse, management, and upload flows with mock inventory and drop-zone previews for collectible intake.

Admin Insights

Dashboard tiles demonstrate stats, recent activity, and authentication prompts using safe placeholder data.

How to Reproduce Locally

1) Install

Run npm install with Node 18+. The project ships with Tailwind CSS, Prisma, and Radix UI components.

2) Static Build

Set NEXT_PUBLIC_USE_STATIC_DATA=true and run npm run build && next export (export emits to out/).

3) Publish

Push the out/ folder to GitHub Pages (or copy into docs/) to share a zero-backend preview.
